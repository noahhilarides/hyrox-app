## AER-001 — Easy Run 45

- **Category:** running
- **Calendar type:** run
- **Variant:** easy
- **Difficulty:** beginner
- **Duration:** 45 min
- **Equipment:** running
- **Focus:** aerobic base, easy pacing
- **Tags:** running, zone2, aerobic, base_phase, beginner

**Description:** Steady Zone 2 run — build the aerobic engine that carries you through eight 1 km legs.

**Progression:** Treat this as your weekly anchor easy run. Add 5 min only after two consecutive weeks at RPE 6 with no walk breaks.

**RPE:** Easy jog segments RPE 5–6; walk breaks RPE 3–4.

**Station cues:**
- Stay tall — think "hips forward," not leaning from the waist.
- On 1 km repeats, start 2–3 sec slower than race pace; negative split if you can.
- First 200 m off the rower/ski: quick cadence, then settle — don't sprint the transition.
- Arms relaxed below chest; drive the knee, don't overstride.
- If you cannot nose-breathe, slow down — that is your Zone 2 ceiling today.

**Substitutions:** Match the written effort, rest, and rep scheme. Substitutions are for missing equipment — not to make the session easier.

No treadmill needed outdoors. Treadmill: set 1% incline for easy/tempo work. Bike or row: use 1 min hard ≈ 200 m run as a rough swap in mixed sessions.

### Session

#### Warm-up
5 min walk → easy jog · 4×20 sec drills (A-skip, butt kicks)

#### Main set
45 min Zone 2 run

#### Block 2
Conversational pace — nose breathing if possible

#### Cool-down
5 min walk + calf / hip flexor stretch

- **Equipment:** running, sled, rower
- **Focus:** race simulation, pacing, mental toughness
- **Tags:** hyrox_simulation, race_specific, race_pace, sled_push, sled_pull, peak_phase, advanced

**Description:** Full HYROX-style simulation with run + station pairs.

**Progression:** Station brick at race intent — log 1 km run splits. Progress sled load 2.5 kg when run split does not slow >5 sec.

**RPE:** Runs RPE 7–8; stations RPE 8–9 on race-pace simulations; skills days RPE 6–7.

**Station cues:**
- Low hips, arms extended — you are pushing the floor away, not pressing up on handles.
- Short, quick steps; if you stall, reset feet under hips before the next drive.
- Exhale on each push; brace before you move the sled.
- Race tip: first 5 m are setup — don't sprint, then build rhythm.
- Hands low on the rope; sit back and walk backward with purpose.
- Drive elbows to ribs — think row, not upright shrug.
- Take short steps; if rope slackens, you are too upright.
- Keep the sled path straight — one correction beats three zigzags.

**Substitutions:** Match the written effort, rest, and rep scheme. Substitutions are for missing equipment — not to make the session easier.

No sled: heavy walking lunges (20–24 steps) or plate push on turf. Match leg burn, not exact meters.

No sled: heavy bent-over row or chest-supported row, 3×8–10 @ RPE 8, same rest as written sled pulls.

#### Block 3
