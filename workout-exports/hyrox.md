## ENG-001 — Mini HYROX

Category: hyrox | Type: race_sim | Variant: simulation
Difficulty: advanced | Duration: 55 min
Equipment: running, ski_erg, rower
Focus: race pacing, station management
Tags: hyrox_simulation, race_specific, conditioning, running, peak_phase, advanced

Description: Short race-format engine block.

Progression: Full simulation density: progress by −30 sec between-round rest when run splits stay within 10 sec.

RPE: Runs RPE 7–8; stations RPE 8–9 on race-pace simulations; skills days RPE 6–7.

Coaching cues:
  - Stay tall — think "hips forward," not leaning from the waist.
  - On 1 km repeats, start 2–3 sec slower than race pace; negative split if you can.
  - First 200 m off the rower/ski: quick cadence, then settle — don't sprint the transition.
  - Arms relaxed below chest; drive the knee, don't overstride.

Substitutions: Match the written effort, rest, and rep scheme. Substitutions are for missing equipment — not to make the session easier.

No treadmill needed outdoors. Treadmill: set 1% incline for easy/tempo work. Bike or row: use 1 min hard ≈ 200 m run as a rough swap in mixed sessions.

## ENG-002 — Race Simulation

Category: hyrox | Type: race_sim | Variant: simulation
Difficulty: advanced | Duration: 75 min
Equipment: running, sled, rower
Focus: race simulation, pacing, mental toughness
Tags: hyrox_simulation, race_specific, race_pace, sled_push, sled_pull, peak_phase, advanced

Description: Full HYROX-style simulation with run + station pairs.

Progression: Station brick at race intent — log 1 km run splits. Progress sled load 2.5 kg when run split does not slow >5 sec.

RPE: Runs RPE 7–8; stations RPE 8–9 on race-pace simulations; skills days RPE 6–7.

Coaching cues:
  - Low hips, arms extended — you are pushing the floor away, not pressing up on handles.
  - Short, quick steps; if you stall, reset feet under hips before the next drive.
  - Exhale on each push; brace before you move the sled.
  - Race tip: first 5 m are setup — don't sprint, then build rhythm.
  - Hands low on the rope; sit back and walk backward with purpose.
  - Drive elbows to ribs — think row, not upright shrug.
  - Take short steps; if rope slackens, you are too upright.
  - Keep the sled path straight — one correction beats three zigzags.

Substitutions: Match the written effort, rest, and rep scheme. Substitutions are for missing equipment — not to make the session easier.

No sled: heavy walking lunges (20–24 steps) or plate push on turf. Match leg burn, not exact meters.

No sled: heavy bent-over row or chest-supported row, 3×8–10 @ RPE 8, same rest as written sled pulls.

--- SESSION ---

Warm-up:
10 min progressive jog + station prep

Main set:
1000m Run → Sled Push

Block 2:
1000m Run → Sled Pull

Block 3:
1000m Run → Burpees

Block 4:
1000m Run → Row

Block 5:
Pace each run at moderate-high effort; stations at race intent

Cool-down:
15 min very easy jog or walk

## ENG-005 — Ski-Burpee Engine

Category: hyrox | Type: race_sim | Variant: simulation
Difficulty: intermediate | Duration: 45 min
Equipment: ski_erg, running
Focus: ski power, burpee capacity
Tags: conditioning, ski_erg, burpees, engine, build_phase, intermediate

Description: Ski and burpee pairing for upper-lower engine stress.

Progression: Run simulations at race effort (RPE 8–9 on stations, controlled 1 km runs). Log transition times. Progress by shaving rest 15–30 sec or adding one station round before adding load. In build phase, progress one variable at a time (load, reps, or density).

RPE: Mixed sessions: runs RPE 7, stations RPE 7–8 unless noted race pace.

Coaching cues:
  - Long arms at the catch; snap the handle past the knees, then finish with triceps.
  - Breathe out on the pull — avoid holding breath through the drive.
  - Legs do 60% of the work; save arms for the last 250 m of each piece.
  - Damper 5–7 for most HYROX prep; higher is not always faster.
  - Hands under shoulders; chest and thighs touch on the floor.
  - Stand up fully before the jump — no half reps under fatigue.
  - Step-back or box burpees are race-legal when grip and lungs are gone.
  - Find a cadence you can hold for 80 reps — sprinting the first 20 costs the run.

Substitutions: Match the written effort, rest, and rep scheme. Substitutions are for missing equipment — not to make the session easier.

No SkiErg: row the same distance/time at the same RPE. Bike: 2 min moderate ≈ 500 m ski in conditioning blocks.

Step-back burpees or box burpees count — keep full range of motion and the same rep total.

--- SESSION ---

Warm-up:
5 min easy ski + jog

Main set:
4 rounds:

Block 2:
750m Ski @ hard

Block 3:
15 Burpees

Block 4:
400m Run @ moderate

Block 5:
Rest 2 min

Block 6:
Substitute: 500m row for ski

Cool-down:
Easy row 5 min
